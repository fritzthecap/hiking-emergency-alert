When running this application for the first time, you have to go through seven steps. The "Contacts" page is the second. Here you have to input at least your name one contact.

Watch the red validation message on top. Clicking anywhere outside an input field may make it disapear.

Your address and phone number can be used as variables in the text of the e-mail you are going to work out in the next step, so it may be useful to also input it. All these data will get stored and you won't have to do them again.

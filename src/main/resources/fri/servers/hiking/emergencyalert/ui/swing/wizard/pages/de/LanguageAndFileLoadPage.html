Diese Desktop-Anwendung läuft zu Hause, während Sie unterwegs sind. Sie sendet Warn-E-Mails, wenn Sie nicht innerhalb der geplanten Zeit zurück sind. Falls Sie einen Unfall in einem Gebiet ohne Telefonverbindung haben, könnte dies Ihr Leben retten!

Sie benötigen einen E-Mail-Anbieter und ein E-Mail-Konto sowie einen Computer mit Internetverbindung. Außerdem benötigen Sie mindestens einen E-Mail-Kontakt, der im Notfall Hilfe für Sie organisieren kann.

  • Keiner Ihrer Kontakte erhält eine E-Mail, wenn Sie zurück sind und rechtzeitig auf die Schaltfläche „Wieder daheim” klicken.

  • Wenn Sie jedoch zu spät kommen, erhält der erste Kontakt eine E-Mail (mit dem Text, den Sie jetzt eingeben werden).

  • Wenn dieser Kontakt nicht innerhalb einer Stunde antwortet, erhält der nächste Kontakt die gleiche E-Mail, und der erste Kontakt wird benachrichtigt, dass der nächste Kontakt alarmiert wurde. All dies ist natürlich konfigurierbar.

  • Wenn einer der Kontakte auf die Benachrichtigungs-E-Mail antwortet, werden keine weiteren Kontakte benachrichtigt.
  • Die Anwendung verwendet eine eindeutige MAIL-ID, um Benachrichtigungsantworten von Ihren privaten E-Mails zu unterscheiden.

Hier auf der ersten Seite (rechts) können Sie Ihre Sprache auswählen oder eine Wanderungsdatei laden, falls Sie bereits eine Wanderung erstellt haben. Die gewählte Sprache wird zusammen mit den Wanderungsdaten gespeichert.

Wenn Sie diese Anwendung das nächste Mal starten, wird diese Seite höchstwahrscheinlich nicht mehr angezeigt, aber Sie können jederzeit über „Zurück” dorthin navigieren und jede von Ihnen erstellte Wanderung laden. Wanderungen werden standardmäßig im JSON-Format gespeichert in

$HOME/hiking-emergency-alert/hike.json

Sie können dies ändern, indem Sie die Anwendung mit -Dhike.home=IhrVerzeichnis und auch -Dhike.file=MeineWanderdatei.json ausführen.

Dies ist eine Open-Source-Anwendung, die Ihre Daten nicht weitergibt oder stiehlt, obwohl sie eine Internetverbindung benötigt, um Alarm-E-Mails zu versenden und Antwort-E-Mails zu empfangen. Das Passwort Ihres E-Mail-Anbieters wird nirgendwo gespeichert, Sie müssen es bei jedem Wanderungsbeginn manuell eingeben. Sie können den Quellcode auf Integrität überprüfen unter github.com - fritzthecap - hiking-emergency-alert.

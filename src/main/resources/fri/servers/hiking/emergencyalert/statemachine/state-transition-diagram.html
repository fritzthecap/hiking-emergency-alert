State-Transition Diagram

Following table crosses all possible states with all possible events. Horizontal column-headers are states, vertical row headers are arriving events. Cells contain follower states according to the arriving event.

Example: when event COMING_HOME arrives in state OnTheWay, the automaton changes to state HomeAgain.

↓ Event		 HikerRegistered HikeActivated OnTheWay OverdueAlert HomeAgain / AlertConfirmed

SETTING_OFF (timer) 🚫 → OnTheWay 🚫 🚫 🚫
COMING_HOME (user) 🚫 → HomeAgain → HomeAgain → HomeAgain 👍
